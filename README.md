# ts-serverless-starter

Starter for Serverless framework + TypeScript.

## usage

```bash
# install all dependencies
$ yarn

# check lint errors with ESLint
$ yarn lint
# fix lint errors with ESLint
$ yarn lint:fix

# type check
$ yarn tsc

# run tests with Jest
$ yarn test

# start developing with offline mode
$ yarn dev
```
